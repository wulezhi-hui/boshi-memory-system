## 四、改造内容

### 4.1 架构拆分：怪兽类→3个小类

```
改造前:                         改造后:
TieredMemory(30方法/660行)      TieredMemory(薄层编排 ~300行)
                                 ├── HotManager    (热区管理)
                                 ├── WarmStore     (温区存储)
                                 └── ColdArchive   (冷区归档)
```

### 4.2 质量修复
- ruff --fix 自动修复 105 个问题
- 3处 bare except 改为 `except Exception:`（broadcast_poller.py:32, chroma_memory.py:48/184）
- 1处E402导入位置问题跳过（sys.path后import，故意为之）

### 4.3 历史修复（本次之前已做）
- `store_conversation()` 长度阈值 `>10`→`>2`，修复短消息漏存
- boshi provider `sync_turn` 增加 `push_live_turn` 调用，支持跨入口实时可见

## 三、核心模块设计

### 3.1 版本链/不可变日志

**原则：** 从不覆盖写入。每次"更新"都是追加新版本 + 标记旧版本 isLatest=false。

```
写入流程：
  旧记忆 (isLatest=true)
    → deprecate_memory(id, superseded_by=new_id)
    → 旧记忆 isLatest=false, _superseded_by=new_id, _deprecated_at=timestamp
    → add_memory(新内容, metadata={isLatest:true, _parent_id=旧id, _version_created=timestamp})

检索流程：
  search_memory(query) → 默认添加 where={"isLatest": True}
  search_memory(query, all_versions=True) → 返回所有版本（包括历史）
```

### 3.2 用户画像层

```
user_profile.py
├─ get_profile(user_id) → {static: [...], dynamic: [...]}
│   静态层：长期偏好/身份（type=profile_static）
│   动态层：近期活动上下文（type=profile_dynamic）
├─ update_profile(user_id, facts) → 自动分类写入
│   偏好/事实 → static
│   任务状态/决策 → dynamic
├─ format_for_prompt() → 生成 system prompt 注入文本
├─ refresh_dynamic_profile() → 热度衰减 + 时间过期清理
└─ 10min 缓存机制（_load_cache / _save_cache）
```

### 3.3 知识图谱

```
knowledge_graph.py
├─ add_relation(from_id, to_id, rel_type) → 关系边 CRUD
│   rel_type: updates | extends | derives | related
├─ get_relations(memory_id) → 查询关联关系
├─ trace_version_chain(memory_id) → 双向追溯版本演化
├─ merge_duplicates(threshold) → 合并重复记忆
├─ add_derived_fact(sources, content, confidence) → 推理事实
│   置信度 < 0.6 自动拒绝
└─ auto_link_entities(user_content, assistant_content) → 自动实体关联
    已集成到 sync_turn 每轮对话自动执行
```

### 3.4 混合搜索

```
hybrid_search(query) → {
  "memories": [...],    // ChromaDB 语义向量结果 + isLatest 过滤
  "sessions": [...],    // state.db SQLite 全文会话结果
  "source": "hybrid"
}
```

### 3.5 矛盾消解 + 自动遗忘

```
auto_forget(dry_run=False) → 扫描 forget_after 字段，过期即标记 isLatest=false
detect_conflicts(threshold=0.15) → 同类记忆 + 极高相似度的潜在冲突对
resolve_conflict(winner_id, loser_id) → 版本化消解
```

## 四、数据流

```
用户消息 → sync_turn
  ├─ extract_facts（三层提取）
  │    ├─ 关键词匹配（60+ 模式）
  │    ├─ 消息结构匹配（正则）
  │    └─ 长度兜底
  ├─ 写入 ChromaDB（conversation_turn + raw_conversation）
  ├─ update_profile（自动画像维护）
  ├─ auto_link_entities（知识图谱关联）
  └─ _auto_worklog（代码变更自动日志）

检索路径：
  Hermes 内部：
    boshi_search("查询")
      → search_memory（语义 + isLatest 过滤 + 降级防护）
      → hybrid_search（语义 + 全文合一）
      → 返回结果

  外部Agent / CLI（v6.1 新增）：
    MCP: tools/call boshi_search
    CLI: boshi search "查询"
      → boshi_core.search()
        → chroma_bridge.hybrid_search / search_memory / knowledge_graph
      → 返回结果
```
